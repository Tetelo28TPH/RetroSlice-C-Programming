# RetroSlice
The RetroSlice Arcade Management System is a C# console-based application designed to manage and analyze customer participation in activities at RetroSlice. The system captures applicant details, evaluates eligibility for arcade game token credit, and tracks statistics related to bowling scores, arcade rankings, and food consumption.

This project demonstrates the use of object-oriented programming (OOP) concepts in C#, including classes, collections, enums, loops, and user input handling within a menu-driven console application.

**1. Applicant Registration**
Users can capture applicant details such as:
Name
Age
Arcade score ranking
Bowling score
Number of pizzas eaten
Slush flavour preference
Number of slushies consumed
Employment status
Allergy information

**2. Applicant Management**
The application allows users to:
Capture new applicant details
Display stored applicant records
Sort applicants by age
View youngest and oldest applicants

**3. Game Token Credit Qualification**
The system determines whether a customer qualifies for RetroSlice game token credit based on several conditions such as:
Age
Employment status
Parents’ employment (if under 18)
Years as a RetroSlice customer
Number of pizzas consumed
Bowling or arcade score
Slush flavour preference

**4. Arcade and Bowling Statistics**
The application calculates and displays:
Highest bowling score
Highest arcade ranking
This provides insight into the top-performing customers.

**5. Pizza Consumption Analytics**
The system can:
Calculate the average number of pizzas consumed
Determine whether a customer qualifies for pizza-based loyalty offers
Customers who have eaten 100 or more pizzas may qualify for discounts.

**7. Console Animations**
The program includes simple console animations such as:
A RetroSlice welcome animation
A loading effect to simulate processing

**Technologies Used**
Language: C#
Framework: .NET Console Application
Concepts Demonstrated:
Object-Oriented Programming
Collections (List<T>)
Enums for menu navigation
Loops and conditional logic
Console input/output
Basic sorting algorithms (Bubble Sort)
